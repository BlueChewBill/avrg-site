# AVRG

Site + assets for AVRG (handmade fingerboards).

## Layout

- `site/` — current live site source (static HTML/CSS/JS, data in `data.js`)
- `redesign/` — redesign explorations, lettered `a-` through `i-` (`h-blackout.html` is the direction being polished; `i-pasteup.html` is the zine/paste-up skeleton; `header-playground.html` is an interactive tab-style explorer)
- `deploy/` — deployable snapshot
- `build_site.py` / `seed_boards.py` — build and data-seeding scripts (Python)
- `boards/` — board photography, organized by line (classic, hand-shaped, originals, resale)
- `completes/` — complete-setup photography
- `clean shots/` — raw/clean product shots
- `AVRG brand img/` — brand imagery

## Notes

Media is committed directly (no LFS); largest files are well under GitHub's 100MB limit.
